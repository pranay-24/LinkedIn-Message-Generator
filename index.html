LinkedIn Connection Message Generator

LinkedIn Connection Message generator is an app to generate text messages which could be copied and used on LinkedIn to send connection requests.First the users enter their own information like name, college name,program name and save it. Then from the options ,they can choose which type of message they want to generate. The user can hover over options to know more about them. Text utility tools like clear,copy and trim white spaces is also provided.
